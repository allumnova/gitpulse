GitPulse is a sleek CLI tool that gives you a quick "pulse check" of your GitHub activity. It visualizes your contribution streak, top languages, and recent activity directly in your terminal.

## Features
- **Streak Tracker**: See your current and longest contribution streaks.
- **Language Pulse**: Breakdown of your top programming languages.
- **Recent Activity**: A quick list of your latest commits and PRs.
- **Terminal Friendly**: Beautiful output with ASCII icons.

## Installation
```bash
npm install -g .
```

## Usage
```bash
gitpulse --check
```

## Configuration
Requires a `GITHUB_TOKEN` in your environment variables.
